# 线性回归算法简介

线性回归是通过属性的线性组合来进行预测的函数，即

​ f(x)=w1x1+w2x2+...+wnxn+bf(x) = w_1x_1 + w_2x_2 + ...+ w_nx_n + bf(x)=w1​x1​+w2​x2​+...+wn​xn​+b ，使得 f(xi)≃yif(x_i) \simeq y_if(xi​)≃yi​

用向量形式表示为

​ f(x)=wTx+bf(x) = w^Tx + bf(x)=wTx+b

提示

离散属性的处理：若有序，则连续化；否则，转化为 K 维向量

# 广义线性模型

对数线性回归的公式如下：

​ ln⁡y=wTx+b\ln{y} = w^Tx + blny=wTx+b

它实际上是在试图让 ewTx+be^{w^Tx + b}ewTx+b 逼近 yyy .形式上仍是线性回归，但实质上在求取输入空间到输出空间的非线性函数映射，更一般地，考虑单调可微函数 g(x)g(x)g(x) ，令

​ y=g−1(wTx+b)y = g^{-1}(w^Tx + b)y=g−1(wTx+b)

这样的模型称作广义线性模型，其中函数g(x)被成为联系函数。显然当联系函数g(x)=ln(x)时，便是对数线性回归。
