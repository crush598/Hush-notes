# Logistic 回归算法简介

逻辑回归是机器学习中的经典分类问题，属于对数线性模型，又称对数几率回归。Logistic 回归可以看作预测值为 “标签的对数几率” 的线性回归模型

对数几率函数

​ 一种 “Sigmoid 函数”，优点单调可微，任意阶可导。

​ y=11+e−zy = \frac{1}{1 + e^{-z}}y=1+e−z1​ 即 ln⁡y1−y=wTX+b\ln{\frac{y}{1-y}} = w^TX + bln1−yy​=wTX+b 反映了 x 作为正例的相对可能性
