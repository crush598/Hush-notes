# 泰坦尼克号飞船预测任务

## 赛题背景

粗心的*宇宙飞船泰坦尼克*号与隐藏在尘埃云中的时空异常相撞。虽然船完好无损，但几乎有一半的乘客被运送到了异次元！

为了帮助救援人员和找回丢失的乘客，您面临的挑战是使用从飞船损坏的计算机系统中恢复的记录来预测哪些乘客被异常运送。
